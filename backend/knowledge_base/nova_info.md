# Nova Agent — Base de Conocimiento

Nova es un asistente virtual inteligente de proposito general.

## Capacidades

- **Conversacion**: Nova puede mantener conversaciones fluidas en multiples idiomas.
- **Analisis de documentos**: Puede procesar PDFs, archivos Markdown y texto plano para ampliar su conocimiento.
- **Vision**: Puede analizar imagenes que el usuario adjunte — fotos, documentos escaneados, graficas, etc.
- **Memoria**: Recuerda informacion relevante de conversaciones anteriores para dar respuestas mas personalizadas.
- **Busqueda RAG**: Utiliza busqueda semantica sobre su base de conocimiento para dar respuestas precisas y basadas en datos.

## Como usar Nova

1. **Chatea** con Nova escribiendo tu pregunta o mensaje.
2. **Sube documentos** (PDF, Markdown, TXT) para que Nova aprenda de ellos y pueda responder preguntas sobre su contenido.
3. **Adjunta imagenes** para que Nova las analice y te de informacion sobre lo que ve.
4. **Revisa conversaciones pasadas** desde el panel lateral.

## Personalizacion

Puedes agregar mas archivos `.md` o `.txt` a la carpeta `knowledge_base/` para que Nova los ingeste automaticamente al iniciar.
